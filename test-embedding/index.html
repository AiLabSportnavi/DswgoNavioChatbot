Navio Widget — Embed Test

This static page pretends to be a SportNavi club website. The widget below is loaded from our deployment exactly the way a real club would embed it. Look for the chat launcher in the bottom-right corner.

What this page does

It includes a single <script> tag that pulls navio-widget.js from the deployed frontend and points it at the deployed backend via data-api. Nothing else on this page is the widget — it mounts itself into an isolated Shadow DOM.

The embed snippet (what a club pastes)

<script
  src="https://navio-frontend-34ktkxptbq-ey.a.run.app/navio-widget.js"
  data-api="https://navio-34ktkxptbq-ey.a.run.app">
</script>

Why it works here

The backend CORS allowlist accepts any *.sportnavi.de origin plus localhost. That is why this page must be opened through a local server (e.g. http://localhost:8000) and not as a file:// path — a file:// origin is null and the backend will reject it. See README.md for the one-line command to serve it.
